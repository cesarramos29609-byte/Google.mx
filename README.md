Inicialización del Cliente de Gemini (GeminiRepository.kt)Crea un repositorio o clase de servicio para interactuar de forma segura con el modelo de Gemini utilizando tu clave de API (almacenada preferiblemente de forma segura en local.properties).Kotlinpackage com.iogeminis.app.dataimport com.google.ai.client.generativeai.GenerativeModelimport com.google.ai.client.generativeai.type.contentclass GeminiRepository { // Inicializa el modelo (por ejemplo, gemini-1.5-flash para respuestas de baja latencia) private val generativeModel = GenerativeModel( modelName = "gemini-1.5-flash", apiKey = "TU_API_KEY_AQUI" // Reemplazar o leer desde BuildConfig ) suspend fun sendMessage(prompt: String): String { return try { val response = generativeModel.generateContent(prompt) response.text ?: "No se obtuvo respuesta." } catch (e: Exception) { "Error al conectar con Gemini: ${e.localizedMessage}" } }}3. Creación del ViewModel (MainViewModel.kt)Maneja el estado de la interfaz y la comunicación en segundo plano utilizando corrutinas de Kotlin.Kotlinpackage com.iogeminis.app.uiimport androidx.lifecycle.ViewModelimport androidx.lifecycle.viewModelScopeimport com.iogeminis.app.data.GeminiRepositoryimport kotlinx.coroutines.flow.MutableStateFlowimport kotlinx.coroutines.flow.StateFlowimport kotlinx.coroutines.flow.asStateFlowimport kotlinx.coroutines.launchclass MainViewModel : ViewModel() { private val repository = GeminiRepository() private val _uiState = MutableStateFlow<String>("¡Hola! ¿Cómo puede ayudarte iOGeminis hoy?") val uiState: StateFlow<String> = _uiState.asStateFlow() private val _isLoading = MutableStateFlow(false) val isLoading: StateFlow<Boolean> = _isLoading.asStateFlow() fun onPromptReceived(prompt: String) { viewModelScope.launch { _isLoading.value = true _uiState.value = "Pensando..." val response = repository.sendMessage(prompt) _uiState.value = response _isLoading.value = false } }}4.
